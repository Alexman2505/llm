# 🌤️ Weather AI Agent

Локальный AI-агент для получения актуальной погоды с использованием Ollama и современного Python-стека.

## ✨ Особенности

- 🤖 **Полностью локальный** — не требует API-ключей и интернет-соединения для работы LLM
- 🌡️ **Бесплатный источник погоды** — данные с wttr.in без ограничений
- 🐍 **Современный стек** — Python asyncio, HTTPX, Agno
- 🧠 **Tool calling** — агент сам решает, когда вызывать функцию погоды
- 🇷🇺 **Поддержка русского языка** — модель отвечает на русском

## 🏗️ Архитектура
Пользователь → AI Агент (Ollama + llama3.2) → wttr.in API → Ответ
↓
get_weather() tool

text

## 📋 Требования

- **Python 3.10+**
- **Ollama** с установленной моделью (рекомендуется `llama3.2:3b`)
- **8GB+ RAM** (4GB для модели + запас)

## 🚀 Быстрый старт

### 1. Установка Ollama

```bash
# Windows / Linux / macOS
curl -fsSL https://ollama.com/install.sh | sh
```
Или скачайте с официального сайта: https://ollama.com

2. Скачивание модели
```bash
ollama pull llama3.2:3b
```
3. Клонирование репозитория
```bash
git clone https://github.com/YOUR_USERNAME/weather-ai-agent.git
cd weather-ai-agent
```
4. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
```
5. Установка зависимостей
```bash
pip install -r requirements.txt
```
6. Запуск агента
```bash
python weather_agent.py
```

## 🛠️ Как это работает
Агент получает ваш вопрос

Понимает, что нужна информация о погоде

Автоматически вызывает функцию get_weather(city)

Получает данные с wttr.in (бесплатно, без ключа)

Формирует естественный ответ на русском языке

## 🔧 Настройка
Смена модели
В файле weather_agent.py измените:
model=Ollama(id="llama3.2:3b")  # Замените на другую модель

Доступные модели с поддержкой tool calling:

llama3.2:3b (рекомендуется, 2GB)

qwen2.5:3b (лучше знает русский, 2GB)

phi3:mini (легкая, 2GB)

##  Смена источника погоды
Замените функцию get_weather() на другой источник:

OpenWeatherMap (требует API-ключ)

python-weather (альтернативный бесплатный парсер)

WeatherAPI.com (40k запросов/мес бесплатно)
